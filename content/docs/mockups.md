# Mockups & UI Notes


This gallery captures the latest iteration of our paper mockups, now digitized to document the UX direction for each persona. Every screen reflects feedback from usability sessions and aligns with the requirements and personas described elsewhere in our journey.

---

## System Admin (Platform Operations)

### Tenant Management workspace
![Tenant management workspace for system admins](/assets/docs/mockups/tenant-management-admin.jpg)

- Centralizes tenant onboarding, status, and feature toggles.
- Quick filters surface at-risk tenants (e.g., overdue invoices, storage limits).
- Inline actions let admins jump into impersonation mode or suspend access.

### Audit logs browser
![Audit logs console for platform operations](/assets/docs/mockups/auditlogs-admin.jpg)

- Chronological log grouped by entity (tenant, user, plugin).
- Pill filters refine by action type (create / update / suspend / delete).

### Support ticket triage
![Ticket triage view for admins](/assets/docs/mockups/tickets-admin.jpg)

## Organization Manager

### User management overview
![Organization manager user management overview](/assets/docs/mockups/user-managment-orgmanager.jpg)

- Department-based grouping with awareness “heat map”.
- Bulk actions assign courses, nudge reminders, or flag for coaching.
- CSV import status with per-user validation feedback.

### Awareness events calendar
![Security awareness event calendar for organization managers](/assets/docs/mockups/event-calender-orgmanager.jpg)

- Unified schedule for phishing campaigns, workshops, and deadlines.
- Drag-to-reschedule interaction updates invitations in real time.

### Learner statistics detail
![Organization manager learner statistics drill-down](/assets/docs/mockups/learner-stats-orgmanager.jpg)

- Timeline view mixes per user phishing outcomes with course completion metrics.
- Risk flags call out repeated offenders and suggested remediation.
- Export strip prepares PDF snapshot for leadership status updates.

## Learners (End Users)

### Personal stats dashboard
![Learner dashboard overview](/assets/docs/mockups/dashboard-learner.jpg)

- User's 'Risk-Rating'.
- Surface certificates expiring soon alongside recommended refreshers.
- Calendar snippet highlights upcoming simulations or workshops.

### Course catalog
![Learner course catalog](/assets/docs/mockups/courses-learner.jpg)

- All available courses (to the user)

### Course overview + module navigation
![Learner course overview page](/assets/docs/mockups/course-overview-learner.jpg)

- High-level description, learning objectives, and module list.
- Module cards show media type (video, quiz, lab) and estimated effort.
- Pre-requisite warning ensures sequencing for learning paths.

### Module player
![Learner module player interface](/assets/docs/mockups/module-content-learner.jpg)

- Split reading pane and notes area support active learning.
- Inline knowledge checks with immediate feedback and remediation links.
- Sticky progress bar keeps track of module and course completion.

### Compliance check-in
![Learner compliance check-in workflow](/assets/docs/mockups/compliance-learner.jpg)

- Step-based wizard for mandatory compliance attestations.
- Contextual help bubble explains policies and acceptable use examples.
- Submission summary emailed automatically to organization manager.
